# **命名技巧**

## **某团队对于repository层的约定**

插入：add()  addAll()

更新：save()  saveAll()

删除：remove()  removeAll()

聚合根查询：find()  findAll()

关键词检索：searchXX()

命名技巧

## **方法名**

动词开头：如createXXX()

避免混用：query/fetch/find/search

规避特殊词：一般方法避免get/set开头


# **变量**

## **1、尽量避免使用单字母或缩写**

即使在for循环中：pos/index也比i/j/k要优雅

## **2、变量名长度适中**

maxAudienceNum

limitOfAudienceNumInstatium //舒适区

## **3、用词技巧**

1、布尔变量

status   //bad

isDone  //good

2、静态变量

flag  //bad

dataReady //good

3、常量

int VERSION = 410

4、集合变量

List<Student> students
